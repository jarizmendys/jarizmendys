# 📝 Visual Studio CODE

## 1. 📚 Rutas de Aprendizaje y Configuración del Editor (VS Code)

Como núcleo del ecosistema de desarrollo, **Visual Studio Code** actúa como el orquestador principal. Para habilitar las capacidades de Inteligencia Artificial y sincronización de entorno, es imperativo realizar el inicio de sesión vinculando una cuenta de **GitHub**.

### 🔌 Extensiones Esenciales

Para un entorno de ingeniería de software optimizado, instale el siguiente stack de extensiones:

- **Spanish Language Pack:** Localización de la interfaz del IDE.
- **GitHub Copilot Chat:** Interfaz conversacional para asistencia en la generación de código.
- **Live Preview:** Renderizado en tiempo real de archivos HTML desde el propio editor.
- **Python:** Soporte integral para desarrollo en lenguaje Python.
- **WSL:** Bridge técnico para abrir directorios y ejecutar binarios dentro del Subsistema de Windows para Linux.
- **Docker:** Suite de herramientas para la gestión de imágenes y contenedores.

### ⌨️ Atajos de Teclado Críticos

La eficiencia operativa en el IDE se fundamenta en el dominio de los siguientes *shortcuts*:

| Acción                            | Combinación de Teclas |
| --------------------------------- | --------------------- |
| Guardar Archivo                   | `Ctrl + S`            |
| Apertura de Terminal Integrada    | `Ctrl + ñ`            |
| Conmutar Barra Lateral Secundaria | `Ctrl + Alt + B`      |
