# 🤖 Ecosistema de Inteligencia Artificial (IA)

## 1. 🧠 Disciplinas de la IA

- **Machine Learning (ML):** Ciclo de Datos -> Entrenamiento -> Evaluación -> Deploy.
- **Deep Learning:** Arquitecturas basadas en Redes Neuronales.
- **NLP:** Procesamiento de Lenguaje Natural.
- **Visión por Computador:** Análisis de información visual.

## 3. 💻 Stack Tecnológico Categorizado

- **Lenguajes:** Python, JavaScript (JS), Node.js.
- **Frameworks:** Django, FastAPI, React.
- **Arquitectura y Patrones:** APIs REST, Microservicios.
- **Bases de Datos:** PostgreSQL, SQLite.
- **Herramientas de Datos:** Notebooklm.

## 4. 📖 Glosario Técnico

- **Contenedor:** Unidad estándar de software que empaqueta el código y todas sus dependencias para que la aplicación se ejecute de forma rápida y fiable en cualquier entorno informático.
- **Bash:** Acrónimo de *Bourne-Again Shell*. Es el intérprete de comandos (shell) estándar para sistemas Linux, encargado de procesar las instrucciones del usuario hacia el núcleo del sistema operativo.
