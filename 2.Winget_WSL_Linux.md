# #🚀 Winget + WSL (Windows Subsystem for linux) + Linux

## 1. 💻Gestión de Windows Terminal (winget)

Administre la terminal nativa de Windows con el gestor de paquetes de sistema:

```bash
- wt -v                                         # Versión de la aplicación (Windows Terminal).
- winget list                                   # Listar Aplicaciones Instaladas.
- winget list Nom                               # Listar los que inician por Nom
- winget uninstall --id Nom                     # Desintalar Aplicacion Nom (--force)
- winget updgrade                               # Ver Actualizaciones Disponibles.
- winget upgrade --all                          # Actualización masiva de dependencias.
- winget install --id Microsoft.WindowsTerminal # Instalación Win.terminal.
- winget install marktext                       # Instalación Marktext
- winget upgrade Microsoft.WindowsTerminal      # Actualización Win.terminal.
```

### 🐧Instalación y Gestión de WSL

```bash
- wsl --install             # Instalar WSL
- wsl --install -d Ubuntu   # Instalar distribución Ubuntu.
- wsl --unregister Ubuntu   # Desistalación de una distribución.
- wsl -v                    # Verificar si se tiene inslado WSL.
- wsl -l -o                 # Lista de distribuciones disponibles .
- wsl -l -v                 # Listado de distribuciones instaladas y Ver.kernel.
- wsl --Update              # Actualización del núcleo de Linux en Windows.
```

- **Tips Operativo:** `Use Ctrl + Shift + V` para pegar texto y tokens dentro de la terminal Linux.

## 2. 🐧 Comandos Esenciales de Linux

Estructura de comandos para la administración del shell:

⚙️ **Administración y Sistema:**

| Comando                 | Propósito Técnico                                             |
|:----------------------- |:------------------------------------------------------------- |
| `sudo apt update`       | Actualizar la lista de paquetes del Sistema Linux.            |
| `sudo apt upgrade`      | Aplicar todas las actualizaciones de paquetes disponibles.    |
| `whoami`                | Identificador del usuario con la sesión activa.               |
| `man [comando]`         | Acceso al manual oficial de la herramienta seleccionada.      |
| `ps aux` / `htop`       | Visualización de procesos; `htop` ofrece una interfaz visual. |
| `sudo apt install htop` | Instalación `htop` permite ver el uso de CPU y RAM.           |

📁 **Navegación y Archivos:**

```bash
- pwd                       # Muestra ruta absoluta actual.
- ls -a                     # Listar todos los archivos.
- ls -la                    # Lista detallada incluyendo archivos ocultos.
- ls -la -h                 # Tamaño del archivo.
- ls -l && ll               # Listar Archivos los visibles.
- cd /                      # Estructura inicial.
- cd ..                     # Retroceder un nivel en el árbol de directorios.
- mkdir                     # Crear Carpeta.
- mkdir -p ./dr1/dr2        # Crear de directorios y subdirectorios.
- touch archivo.txt         # Crear un archivo vacío.
- cp arch.txt cop.txt       # Copiar archivo a copia
- cp arch.txt /home/us/     # Copiar archivo a Carpeta
- cp -r carp/ copia_carp/   # Copiar Carpetas
- rm archivo.txt            # Eliminar archivo.
- mv [orig] [dest]          # Mover o renombrar.
- cat saludo.txt            # Mostrar contenido archivo.
- echo "hola" > saludo.txt  # Crear Archivo y Contenido.
```

🛠️ **Utilidades y Disco:**

```bash
- df -h               # Reporte de espacio en disco Legible.
- du -sh              # Cuanto pesa una cartepa especifica ./carpeta
- clear o Ctrl + L  # Limpieza del buffer de pantalla.
- ip a                # Direccion IP y estado de red.
- history             # Historial de Comandos Usados
- code .              # Comando Abrir Visual Studio Code desde el directorio actual.
```

### 🐍 Entorno Python (PIP) y Dependencias

Para un entorno de desarrollo profesional, se deben instalar dependencias de compilación del sistema antes de usar el gestor de paquetes de Python:

1. **Instalación de PIP:** `sudo apt install -y python3-pip`.
2. **Dependencias de Compilación:** `sudo apt install -y build-essential libssl-dev libffi-dev python3-dev`.
3. **Verificación:** `python3 -V` y `pip -V`.
4. **Ejecución de Scripts:** `python3 script.py`.
